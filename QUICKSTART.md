# Quick Start Guide

## Get Your Site Running in 3 Minutes

### Step 1: Install Zola

Pick your platform:

**Mac:**
```bash
brew install zola
```

**Windows (with Chocolatey):**
```bash
choco install zola
```

**Windows (with Scoop):**
```bash
scoop install zola
```

**Or download directly:** https://github.com/getzola/zola/releases

### Step 2: Run Your Site

1. Open Terminal/Command Prompt
2. Navigate to this folder:
   ```bash
   cd path/to/erin-website
   ```
3. Start the development server:
   ```bash
   zola serve
   ```
4. Open your browser to: **http://127.0.0.1:1111**

That's it! Your site is running locally.

### Step 3: Make Changes

- **Edit content:** Open `content/_index.md` in any text editor
- **Change colors:** Edit `static/style.css`
- Save the file and your browser will auto-reload!

## Need Help?

- **Zola won't install?** Try downloading the pre-built binary from the releases page
- **Site won't load?** Make sure you're in the `erin-website` folder when you run `zola serve`
- **Want to change styling?** All CSS is in `static/style.css` - it's well-commented!
